Bases: SumLinearOp

Parameters:

linear_ops – The Linear Op and the DiagLinearOp to add to it.

below is copy paste of AddedDiagLinearOperator from the linear_operator library in github.

A SumLinearOperator, but of only two linear operators, the second of which must be a DiagLinearOperator.

param linear_ops:

The LinearOperator, and the DiagLinearOperator to add to it.

param preconditioner_override:

A preconditioning method to be used with conjugate gradients. If not provided, the default preconditioner (based on the partial pivoted Cholesky factorization) will be used (see Gardner et al., NeurIPS 2018 for details).
